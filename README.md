# Django Blog Website

A fully functional, responsive blog website built with **Django** — a high-level Python web framework — featuring user authentication, CRUD operations for posts and comments, and a clean, modern design.

---

## Features

- User registration, login, and logout functionality
- Create, read, update, and delete (CRUD) blog posts
- Add, edit, and delete comments on posts
- Responsive design for desktop and mobile devices
- Rich text editor for writing posts
- User profile with avatar upload
- Admin dashboard for site management

---

## Technologies Used

- Python 3.13.3
- Django 6.0
- SQLite (default) / easily switchable to PostgreSQL or MySQL
- HTML5, CSS3 for frontend styling
- Django’s built-in authentication system
- Django Crispy Forms (optional)
